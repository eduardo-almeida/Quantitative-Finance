2. Aprendizado de máquina em finanças
O aprendizado de máquina tem sido usado em finanças há anos. Aqui está um exemplo da Zacks, que usa aprendizado de máquina para prever os lucros futuros das empresas. Um relatório do JP Morgan diz que os investidores que não conhecem o aprendizado de máquina serão deixados para trás.

3. Preveja o futuro com ML
A maioria dos dados que usaremos são preços de ações, como o preço das ações da AMD mostrado aqui. Usaremos muitos métodos de aprendizado de máquina para prever preços futuros e selecionar carteiras de ações.

4. Mostre-me os dados
Neste curso, usaremos alguns estoques, incluindo LNG e AMD. Eles são altamente voláteis, o que significa que seu preço se move muito.

5. Compreender os dados
Os dados são armazenados em DataFrames do pandas, como amd_df. Incluí apenas duas colunas que usaremos como entradas em nossos algoritmos de aprendizado de máquina - o preço de fechamento ajustado e o volume ajustado. O preço de fechamento ajustado é o preço no final de cada dia, ajustado para itens como desdobramento de ações. O volume ajustado é o número de ações trocadas naquele dia.

6. Compreender os dados
Sempre queremos fazer análise exploratória de dados, ou EDA, em novos dados para entendê-los. O EDA é amplo, mas vamos nos concentrar em gráficos de dados brutos, histogramas, gráficos de dispersão e correlações.

7. Gráficos EDA
A biblioteca do pandas possui alguns métodos de plotagem embutidos para DataFrames. Por exemplo, podemos usar dot-plot () para mostrar gráficos de linha dos dados brutos e plot-dot-hist () para histogramas. O uso de aprendizado de máquina para finanças pode ser realizado de várias maneiras. Poderíamos prever os preços brutos de nossas ações, mas normalmente preveremos as variações percentuais. Isso facilita a criação de um modelo de uso geral para a previsão do preço das ações. Também tornará mais fácil interpretar nossos resultados.

8. Mudanças de preço
Para obter a variação percentual ao longo de um período, podemos usar dot-pct_change () em um DataFrame do pandas. Podemos usar isso para ver um histograma das mudanças percentuais ao longo do tempo, o que normalmente é uma distribuição quase normal enviesada para a direita para ações.

9. Dados de turno
Outra ferramenta que usaremos é shift () dos pandas. Isso muda o índice, portanto, se dermos -10 conforme mostrado aqui, criaremos uma nova coluna com o preço de fechamento em 10 dias no futuro. Podemos então obter a variação percentual do preço em 10 dias no futuro com o ponto-pct_change () do pandas.

10. Correlações
Também veremos as correlações com a função corr () dos pandas. corr () usa correlação de Pearson por padrão, que assume distribuições normais e mede correlações lineares.

11. Correlações
Se você fosse examinar a relação entre o preço atual e o preço futuro, veria uma alta correlação de Pearson. Por exemplo, este gráfico dispersa o preço atual versus o preço 10 dias no futuro e mostra uma correlação de Pearson de cerca de 98%. Essa correlação é grande porque a magnitude bruta dos preços não pode mudar tão rápido em 10 dias. No entanto, isso é enganoso; embora pareça que podemos prever os preços futuros com base no preço atual, é apenas uma miragem. A faixa de preços futuros com base no preço atual é muito grande para ser útil.

12. Correlações
Em vez disso, verificaremos as correlações entre nossas alterações de preços percentuais para ver se as alterações de preços anteriores podem prever alterações de preços futuras. Observe que os limites dos eixos são quase os mesmos porque as alterações de preço são apenas 10 dias de intervalo.
